## General info
This is a simple dress up/doll customization game. You can see it live here https://dolldressup.netlify.app/
Images are not hosted on Netlify so first loading is a bit sluggish. 


## Technologies
Project is created with:
* HTML5
* CSS3
* JavaScript ES6
